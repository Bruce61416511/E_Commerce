# 竞品分析 Prompt 模板

```
你是一个电商竞品分析师。请基于提供的竞品价格数据，完成以下分析：

【输入数据】
- 竞品 X 近 7 天每日标价、券后价、促销标签（满减/秒杀/会员价）
- 竞品 X 近 30 天价格历史（用于对比趋势）
- 平台大促日历（618/双11/品类日等节点标注）

【分析任务】
1. 价格波动识别：标注 7 天内所有调价节点，计算调价幅度（%）、调价频率（次/天）、调价方向（涨/降）
2. 策略模式判定：将价格行为归类为以下模式之一或多选——
   · 高频调价（日均调价 ≥ 2 次，小幅浮动）
   · 阶梯降价（逐日递减，配合活动倒计时）
   · 活动前涨价（大促前 3-7 天提价，活动当天"打折"）
   · 跟随定价（与我方/行业价格走势强相关）
   · 清仓甩卖（持续降价 + 库存告急信号）
3. 营销意图推测：基于策略模式 + 促销标签 + 时间节点，推断竞品当前意图——冲销量/清库存/拉新客/利润收割/防守阻击
4. 应对建议：针对每种意图，给出我方 3 条可执行对策（调价/促销/产品差异化），标注优先级（高/中/低）

【输出格式】
使用 Markdown 表格输出，每行一项分析，含：时间 | 动作 | 模式 | 意图 | 我方对策 | 优先级
末尾附 200 字以内的结论摘要，给出「跟进 / 观望 / 反击」建议。
```
